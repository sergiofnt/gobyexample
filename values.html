Go за Прикладом: Значення (Values)

Go має різні типи значень, включаючи: рядки (strings), ціли числа (int), числа з рухомою комою (float), логічні (bool) абощо. Ось кілька базових прикладів.

 
package main

import "fmt"

func main() {

Рядки можуть бути об’єднані поміж собою за допомогою оператора +.

 
    fmt.Println("go" + "lang")

Цілі числа та числа з плваючою комою.

 
    fmt.Println("1+1 =", 1+1)
    fmt.Println("7.0/3.0 =", 7.0/3.0)

Логічний тип та логічні оператор.

 
    fmt.Println(true && false)
    fmt.Println(true || false)
    fmt.Println(!true)
}

$ go run values.go
golang
1+1 = 2
7.0/3.0 = 2.3333333333333335
false
